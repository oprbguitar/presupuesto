# Presupuesto Perú

Visor ciudadano de ejecución presupuestal del Perú (2022–2026), con datos oficiales del MEF.

**Ver el portal:** https://oprbguitar.github.io/presupuesto/

## Módulos

- **Ejecución presupuestal** (Explorar, Ranking, Fuentes) — datos oficiales del MEF.
- **Contrataciones públicas** — consulta y análisis de contrataciones (OECE/SEACE, MEF, Perú
  Compras, Contraloría). Módulo desacoplado y reversible.
  Ver [docs/contrataciones/README.md](docs/contrataciones/README.md).
